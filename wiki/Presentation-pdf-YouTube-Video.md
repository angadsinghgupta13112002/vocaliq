## What the Presentation Covers

The 13-slide deck walks through:

1. **Title** — VocalIQ branding, team, live URL, GitHub
2. **The Problem** — 75% public speaking anxiety, $100–300/hr coaching cost, zero instant tools
3. **VocalIQ Solution** — 6 key features: record/upload, Gemini AI, emotion timeline, gesture detection, scored report, Drive integration
4. **Technology Stack** — 9 Google Cloud technologies in a 3×3 grid
5. **System Architecture** — Browser → Express API → Google Cloud three-column layout
6. **Three-Pass Gemini Pipeline** — Pass 1 (assessment), Pass 2 (coaching), Pass 3 (gesture analysis)
7. **Authentication** — Google OAuth 2.0 + JWT 7-step flow + security notes
8. **Database** — Firestore `users` and `coaching_sessions` field tables
9. **Coaching Report** — 5 tabs: Overview, Emotion Timeline, Gestures, Transcript, Tips & Coaching
10. **Analytics & Logging** — GA4 frontend + backend Measurement Protocol + Cloud Run logging
11. **Deployment** — Multi-stage Dockerfile + Cloud Run configuration
12. **Lessons Learned** — Express 5 breaking change, PORT env var, ADC vs key files, Vite env baking, GA4 Measurement Protocol, two-pass prompting
13. **Thank You / Q&A**
